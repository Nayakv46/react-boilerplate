React boilerplate project to kickstart your projects!

Project uses commonly used libraries such as: 
- @tanstack/react-query
- react-hook-form
- axios
- shadcn
- zustand
- tailwindcss
  
And much more!

To run it locally simply clone the repo to your local environment using `git clone`

Then use

`cd react-boilerplate`
`npm install`
`npm run dev`
